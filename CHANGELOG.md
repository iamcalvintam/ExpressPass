# Changelog

All notable changes to ExpressPass will be documented in this file.

## [1.0.0] - 2026-03-27

### Added
- Initial release
- Per-app setting profiles with SQLite persistence
- Apply & Launch workflow with automatic setting modification
- Auto-revert via foreground service monitoring `UsageStatsManager`
- 6 built-in setting templates (Developer Options, USB Debugging, Wireless Debugging, Mock Locations, Accessibility Services, Auto-Rotate)
- Home screen shortcuts with deep link support
- Local notifications for setting apply/revert events
- 5-step onboarding wizard for permission setup
- Material 3 UI with dynamic color (Material You) support
- Light and dark theme support
- App grid layout with "Configured Apps" quick access section
- Search filtering across all installed apps
- Settings screen with theme customization and permission management
- Custom Express Pass app icon
